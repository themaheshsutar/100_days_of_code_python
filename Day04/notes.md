# Day 04 - Rock, Paper, Scissors

## Project: Rock, Paper, Scissors Game

A simple command-line game where the user plays Rock, Paper, Scissors against the computer.

## Key Concepts Learned

- **Random module**: Using `random.choice()` to make random selections
- **Lists**: Storing multiple items (game options) in a list
- **Input validation**: Checking if user input is within valid range
- **Conditional logic**: Using if/elif/else statements for game rules
- **Comparison operators**: Checking equality and implementing game logic
- **ASCII art**: Using multi-line strings to create visual representations

## Game Rules

- Rock beats Scissors
- Paper beats Rock
- Scissors beats Paper
- Invalid input results in automatic loss

## Code Structure

1. Define ASCII art for rock, paper, and scissors
2. Store options in a list
3. Get user input (0, 1, or 2)
4. Validate input
5. Generate computer's random choice
6. Compare choices and determine winner
